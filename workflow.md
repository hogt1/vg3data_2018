## Standard workflov
1. Opprett et repository i github
2. **clone** til egen katalog
3. **commit** etter som en gjør endringer 
4. **push** endringene til github
5. **pull** for å hente endringer fra github (**fetch** + **merge**)

 ## Ved merge conflict
 1. Fiks opp i problemer
 2. **stage** filen
 3. **commit** 
 4. **push**
